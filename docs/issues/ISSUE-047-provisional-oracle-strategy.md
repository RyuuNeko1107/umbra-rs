# ISSUE-047: 暫定オラクル戦略（M2 暫定第一義の格上げ・二段ゲート化・accuracy §3.3 改訂指示）

- crate: umbra-fixtures / テスト基盤・横断（全暦/幾何 Issue の受入条件に影響）
- 依存: ISSUE-038（MockEphemeris 実装＝幾何ロジックの足場）, ISSUE-036（JPL DE 第一義オラクル・M10）, ISSUE-029（ゴールデン20・NASA/USNO 第二義）, ISSUE-030（誤差統計・層分解・ToleranceProfile）, ISSUE-012（Ephemeris trait）, ISSUE-001（規約）
- milestone: M2 暦（暫定ゲートの起点）／横断（受入条件として M2〜M9 各 Issue に二段化を波及・最終ゲートは M10）
- モード(tdd-workflow): strict（検証戦略の正本。第一義オラクル（JPL DE=036）が M10 のため、M2〜M9 が「オラクル不在で確定できない」問題を解く方針定義。受入ゲートの二段化は全層の検証根拠を決めるため strict）

## 目的
**第一義オラクル＝JPL DE（ISSUE-036）が M10** であるという制約（accuracy.md §3.1, data-sources §2.3）に対し、**M2〜M9 の暦/幾何 Issue を確定可能にする暫定オラクル戦略**を定義する（milestone0-review §確定D「暫定オラクル戦略」, §Critical「第一義オラクル=JPL DE(036)がM10なのにM2暦確定がDE依存 → 047 暫定オラクル戦略＋036のM2前倒し（最小SPK reader）で二段ゲート化」）。
- **暫定第一義への格上げ（確定D）**: M2〜M9 の期間、**`MockEphemeris`（ISSUE-038）＋ SOFA/IERS 公開テストベクトル ＋ NASA 第二義（ISSUE-029）** を「**暫定第一義オラクル**」に格上げする。
- **二段ゲート化**: 各暦/幾何 Issue の受入を以下の二段に分割する方針を定義する:
  - **M2 暫定ゲート**: Mock（人工配置の幾何健全性）＋ SOFA/IERS テストベクトル（時刻系・歳差章動・恒星時の数値一致）＋ NASA/USNO 整合（慣習を揃えた系統チェック）。
  - **M10 最終ゲート**: JPL DE 差分（ISSUE-036・第一義）で暦誤差を切り出し、保証値を確定（accuracy.md §3.1/§4）。
- **accuracy §3.3 改訂指示**: 「ELP/MPP02・VSOP87D 項数確定手順（Milestone 2）」の「項数確定(M2)」を **「暫定確定(M2)→DE確定(M10)」** へ改訂する（DE 差分が M10 のため、M2 では暫定第一義で**暫定打切り次数**を決め、M10 で DE 差分により最終確定する）。本 Issue はその改訂を accuracy.md に反映させる指示を含む。

## アルゴリズム概要
1. **暫定第一義の確立**: M2〜M9 では MockEphemeris（人工配置で幾何ロジックを単体検証）＋ SOFA/IERS テストベクトル（時刻系・歳差章動・恒星時・ERA の数値一致）＋ NASA/USNO（慣習を揃えた整合）を暫定第一義とする。
2. **二段ゲートのテンプレート化**: 各暦/幾何 Issue の「受け入れテスト」を [M2 暫定ゲート] と [M10 最終ゲート] の二段で記述する雛形を定義。暫定ゲートで暫定 pass、最終ゲートで DE 差分により保証値を確定。
3. **項数の暫定確定（accuracy §3.3 改訂）**: VSOP87D/ELP-MPP02 の打切り次数を M2 では暫定第一義（SOFA/NASA との整合余裕＋Mock 健全性）で**暫定確定**し、`EphemerisMetadata` に「暫定」と明記。M10 で DE 差分により最終確定（残差 太陽0.05″/月0.1″）。
4. **対象 Issue の特定**: 暦・幾何で「DE 差分で最終確定すべき値」を持つ Issue を二段ゲート対象として列挙（下記）。
5. **統計の二段化**: ISSUE-030 に暫定/最終を区別した誤差統計・層分解を要求。
6. **accuracy.md の改訂反映**: §3.3 見出し・手順を「暫定確定(M2)→DE確定(M10)」へ改訂する指示を本 Issue の成果物に含める。

## 許容誤差
- 本 Issue は数値計算を持たないため**固有の許容誤差なし**。
- ゲートの許容: M2 暫定ゲートは「SOFA/IERS テストベクトルとの数値一致（各ベクトルの公称桁）」「Mock 幾何の論理一致」「NASA との慣習整合（系統差を分離報告・絶対基準にしない）」。M10 最終ゲートは accuracy.md §1/§2 の保証値（最大食 ±1〜2s・接触 ±2s・食分 ±0.0005 等）を DE 差分で確定。
