> 出典（一次）: IMCCE 配布 `ftp://ftp.imcce.fr/pub/ephem/moon/elp82b/` の **著者リファレンス Fortran `elp82b_1`**
> （Bureau des Longitudes, MCJCGF.9601, 1996; M. Chapront-Touzé & J. Chapront）＋ `README`。
> 取得日 2026-06-15。これは GPL 再実装（ytliu0 等, 取込禁止）ではなく **著者自身のリファレンス配布**＝
> 「一次原データから自前生成」の原典。本ドキュメントは同 Fortran の**定数・アルゴリズム（科学的事実）**を
> 忠実に転記したもので、コードの移植ではない（係数値の転記は VSOP87/章動と同じ運用）。
>
> 関連: `docs/algorithms/03-ephemeris.md`（§3.2 ELP, 但し MPP02 想定の旧記述）, ISSUE-034（係数取込）, ISSUE-013（太陽の対）。

## 0. 出力仕様

- 出力 = 月の**地心直交座標** `(X, Y, Z)` km。
- フレーム = **平均力学的黄道・J2000 慣性分点**（"mean dynamical ecliptic and inertial equinox of J2000", JD 2451545.0）。
- 入力時刻 = TDB ユリウス日。時間引数 `t = (JD_TDB − 2451545.0)/36525`（**ユリウス世紀**, VSOP の千年とは異なる）。
  `t(1)=1, t(2)=t, t(3)=t², t(4)=t³, t(5)=t⁴`。

## 1. 定数（`elp82b_1` より逐語転記。`rad = 648000/π` 秒角/rad, `deg = π/180`）

```
ath = 384747.9806743165      ! km（距離級数の基準 a）
a0  = 384747.9806448954      ! km（距離スケール: r *= a0/ath）
am  = 0.074801329518
alfa= 0.002571881335
dtasm = 2*alfa/(3*am)
preces = 5029.0966 / rad     ! 歳差定数（秒角/世紀 → rad/世紀）
```

### 1.3 定数補正（DE200/LE200 フィット）

```
delnu = +0.55604/rad/W1(t¹)     ! = 0.55604/rad / 1732559343.73604*rad ... 実装は (0.55604/rad)/w(1,2)
dele  = +0.01789/rad
delg  = -0.08066/rad
delnp = -0.06424/rad/W1(t¹)     ! (−0.06424/rad)/w(1,2)
delep = -0.12879/rad
```

### 1.4 Delaunay 引数 del(j, i)（j=1..4 = D, l', l, F; i=1..5 t べき）

```
do i=1..5:
  del(1,i) = W1(i) − T(i)      ! D
  del(2,i) = T(i)  − ϖ'(i)     ! l'（太陽平均近点離角）
  del(3,i) = W1(i) − W2(i)     ! l （月平均近点離角）
  del(4,i) = W1(i) − W3(i)     ! F （月緯度引数）
del(1,1) += π                  ! D の定数項に +180°
```
歳差/章動系列用引数 `zeta`: `zeta(1)=W1(t⁰)`, `zeta(2)=W1(t¹)+preces`。

### 1.5 J2000 回転（Laskar P,Q 多項式, rad）

```
p1..p5 = 0.10180391e-4, 0.47020439e-6, -0.5417367e-9, -0.2507948e-11, 0.463486e-14
q1..q5 = -0.113469002e-3, 0.12372674e-6, 0.1265417e-8, -0.1371808e-11, -0.320334e-14
```

## 2. 級数評価（ファイル群ごと）

各係数の閾値（`prec`）打切りは**行わない**（全項総和 = prec→0）。`iv = (file−1) mod 3 + 1`（1=経度,2=緯度,3=距離）。
和の単位: 経度/緯度 = 秒角、距離 = km。

### 2.1 主問題（ELP1-3）— **要・係数 6〜7 個**（ISSUE-034 修正必須・後述）

各行 = 4 整数 `ilu(1..4)` + 7 実数 `coef(1..7)`。`coef(1)=A`、`coef(2..7)` = 偏微分（B 列）。
```
tgv = coef(2) + dtasm*coef(6)
if file==3 (距離): coef(1) -= 2*coef(1)*delnu/3
x = coef(1) + tgv*(delnp − am*delnu) + coef(3)*delg + coef(4)*dele + coef(5)*delep
y = Σ_{k=1..5} Σ_{i=1..4} ilu(i)*del(i,k)*t(k)
if iv==3 (距離): y += π/2          ! 余弦化（sin(y+π/2)=cos y）
y = y mod 2π
r(iv) += x*sin(y)
```
**coef(2)..coef(6) を使う**（coef(7) は未使用）。→ ISSUE-034 は主問題で `coef(1)` のみ保存しており**不足**。

### 2.2 地球形状・潮汐・月形状・相対論・惑星(太陽離心率)（ELP4-9, 22-36）— 乗数 5 個

各行 = 5 整数 `iz, ilu(1..4)` + `pha`(度) + `x`(振幅)。
```
if file in 7..9:   x *= t        ! Poisson /t
if file in 25..27: x *= t        ! /t
if file in 34..36: x *= t²       ! /t²
y = pha*deg + Σ_{k=1..2} [ iz*zeta(k) + Σ_{i=1..4} ilu(i)*del(i,k) ] * t(k)
y = y mod 2π
r(iv) += x*sin(y)
```

### 2.3 惑星摂動（ELP10-21, 表1=10-15 / 表2=16-21）— 乗数 11 個

各行 = 11 整数 `ipla(1..11)` + `pha`(度) + `x`。
```
if file in 13..15 or 19..21: x *= t   ! /t
y = pha*deg
表1 (file<16):  Σ_{k=1..2} [ (ipla(9)*del(1,k)+ipla(10)*del(3,k)+ipla(11)*del(4,k))
                              + Σ_{i=1..8} ipla(i)*p(i,k) ] * t(k)
表2 (file>=16): Σ_{k=1..2} [ Σ_{i=1..4} ipla(i+7)*del(i,k) + Σ_{i=1..7} ipla(i)*p(i,k) ] * t(k)
y = y mod 2π
r(iv) += x*sin(y)
```
- 表1: ipla(1..8)=8 惑星, ipla(9)=D, ipla(10)=l, ipla(11)=F（l' は不使用）。
- 表2: ipla(1..7)=7 惑星(Me..Ur), ipla(8..11)=D,l',l,F。

## 3. 最終組立（球面→直交→J2000 回転）

```
V = r(1)/rad + W1(t⁰)+W1(t¹)·t+W1(t²)·t²+W1(t³)·t³+W1(t⁴)·t⁴   ! 経度 rad（W1 平均黄経を加算）
U = r(2)/rad                                                      ! 緯度 rad
R = r(3) * a0/ath                                                  ! 距離 km

x1 = R·cos(U);  x2 = x1·sin(V);  x1 = x1·cos(V);  x3 = R·sin(U)    ! 平均黄道 of date 直交
pw = (p1+p2·t+p3·t²+p4·t³+p5·t⁴)·t
qw = (q1+q2·t+q3·t²+q4·t³+q5·t⁴)·t
ra = 2·√(1−pw²−qw²);  pwqw = 2·pw·qw;  pw2 = 1−2·pw²;  qw2 = 1−2·qw²;  pw·=ra; qw·=ra
X = pw2·x1 + pwqw·x2 + pw·x3
Y = pwqw·x1 + qw2·x2 − qw·x3
Z = −pw·x1 + qw·x2 + (pw2+qw2−1)·x3
```

## 4. オラクル（検証）

- **公開リファレンス値（celestialprogramming, 著者 Fortran 由来）**: JD_TDB 2451555.5 →
  `X=382979.7604730463, Y=-68204.20174530084, Z=-25987.71602589964`（km）。
- 一次アルゴリズムの**別言語移植（Python port of elp82b_1）**を Docker 内で走らせ任意日付の参照値を生成
  （pyerfa と同じ「独立言語実装」位置づけ。上記公開値で port 自体を 1 点アンカー）。
- 追加（M10/絶対精度）: JPL Horizons 地心月位置と比較し ELP82B 固有誤差（~秒角級, `docs/accuracy.md` 月 0.40″）を確認。

## 5. ISSUE-034 への影響（修正必須）

主問題（ELP1-3）の評価は `coef(1..6)`（A＋偏微分 5 列）を使う。ISSUE-034 のパーサ／packed は
**`coef(1)` のみ保存**したため評価器に不足。→ ISSUE-014 の前段スライス（S0）で主問題を**全 7 係数保存**
（`coef(1..7)`、評価器は 1..6 使用）へ修正し packed 再生成。摂動（5/11 乗数 + [pha, x]）は現状で正しい。
