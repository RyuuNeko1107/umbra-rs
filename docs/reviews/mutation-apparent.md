出力 `s = B_geo(t−τ) + (E(t−τ)−E(t))`（一次近似 `−v_E·τ`、SOFA `iauAtciq` の light-time ステップ相当）。

## 生存 7 件（いずれも**不動点反復の機構**に関する等価変異・許容）

生存はすべて「τ の**初期値**」または「**収束打切りの簿記**」に対する変異で、**収束した戻り値を
変えない**。係数符号の生存（遠方エポックで励起し殺せる）と異なり、不動点が不動点である以上
どのエポック・どの天体でも結果が同一になる**真の等価変異**で、原理的に殺せない。
solver.rs の Brent/golden 加速ヒューリスティクス（二分法フォールバックで根は契約どおり）と同じ category。

### A. τ0 初期値 `|B_geo(t)|/c` の `/`→`%`/`*`: line 74:29（2件）
τ0 は反復の初期推定にすぎない。反復 `τ ← |B_geo(t−τ) + (E(t−τ)−E(t))|/c` は強収縮（縮小率
≈ v_E/c ≈ 1e-4 /反復）で**自己補正的**。τ0 を `|B_geo|·c`（≈4.5e13 s）や `|B_geo| mod c` に壊しても、
各反復で τ が ~1e-4 倍に縮小し 4 反復で正しい桁へ回復、5 反復目で収束する（太陽で実トレース確認）。
よって戻り値・τ ともに不変。**τ0 は観測可能量に出ない**ため XYZ/τ テストでは原理的に捕捉不可。

### B. 収束打切り判定 `(next − tau).abs() < 1e-6` の変異: line 81:44（`<`→`==`/`>`/`<=`）, 81:31（`−`→`+`/`/`）（5件）
収束判定は**早期打切りの最適化**であり、停止位置を変えても収束値は変わらない:
- `<`→`==`（float の厳密一致でほぼ不成立）, `<`→`<=`（境界 1e-6 ちょうどは到達不能）, `−`→`+`
  （`next+tau`≈998s, 月≈2.6s で < 1e-6 不成立）, `−`→`/`（`next/tau`≈1.0 で不成立）:
  いずれも `converged` が立たず**常に上限5反復**を回すが、2 反復で収束済みのため戻り値同一。
- `<`→`>`（判定反転）: 初回反復で |Δτ| > 1e-6 が真→`converged=true`→**1 反復で打切り**。だが
  τ0 = |B_geo(t)|/c は既に優秀な初期推定（光行時間中の太陽地心距離変化 ≈175 km ⇒ |τ0−τ1| ≈ 5.8e-4 s、
  2 反復目残差 ≈ 6.8e-10 s）であり、**1 反復で残差は既に < 1e-6 s**。D1/D2（不動点残差 < 1e-6）も
  B1–B3・A1/A2（tol km 級）も 1 反復と 2 反復を区別できない（差は τ で ~1e-9 s, 位置でサブ m）。

numerical-policy §A3 が「固定回数でなく相対収束判定」を要求するため固定反復化による kill は不可
（仕様逸脱になる）。反復回数を契約（戻り値）に出すのは過剰設計。

## 判断
7 件すべて、**不動点反復の初期値・停止簿記に対する等価変異**で、収束した戻り値（位置・τ）を
変えない。物理・契約のコア（第2項の符号と合成、放射時刻、c 除算）は 7 caught で全て捕捉済み。
よって**全件許容**。第2項欠落（本スライスの最重要回帰）は A1/A2・B1/B2/B3・D1/D2 が多重に red にする
ことを red 確認工程（第2項なし実装で当該テスト群が red・他が green）で実証済み。
