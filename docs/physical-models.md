# 物理モデルの具体式 (physical-models)

精度プロファイルから分離した**物理モデルの式と既定パラメータ**を確定する。対象は (C1) 大気差、(C2) 日の出・日没と可視性、(C3) 種別判定の閾値。
conventions.md（§7 方位高度 / §9 半径）・accuracy.md・algorithms.md §8/§9 と整合。これにより ISSUE-023/028 の「要確認」の一部を解消する。

> 状態: ドラフト（Milestone 0）。数値は一次資料で照合済（末尾 出典）。式番号未確証は「要確認」を残す。
> 既定条件: 標準気圧 P0 = 1010 hPa、標準気温 T0 = 10 ℃（大気差の基準）。

## C1. 大気差モデル（`RefractionModel`）

`RefractionModel { None, Standard }`（conventions §7）。既定の高度は**幾何学的（真）高度**。`Standard` で見かけ高度を別に提供する。

### 既定式: Saemundsson（真→見かけ）

幾何学的（真）高度 `h`（度）から見かけ高度を得る。日食では太陽高度は計算で真高度が出るため、**真→見かけ**の Saemundsson を既定とする。

```
R(arcmin) = 1.02 / tan( h + 10.3/(h + 5.11) )        … (C1.1)  h は度
h_app = h + R/60     (度)
```

気圧・気温補正（標準条件からのスケール）:

```
R = R · (P / 1010) · (283 / (273 + T))               … (C1.2)  P:hPa, T:℃
```

- 精度: 補正込みで地平〜天頂で ~0.015′（Saemundsson）。
- 適用範囲: **h ≳ 0**（地平〜天頂）。地平線付近（h≲数°）以下は非保証（accuracy.md §6）。

### 逆変換: Bennett（見かけ→真）

日の出・日没や「見かけ地平通過」を見かけ高度基準で求根する場合に使う。

```
R(arcmin) = 1 / tan( h_app + 7.31/(h_app + 4.4) )     … (C1.3)
h = h_app − R/60
```

同じ (C1.2) の P/T 補正を掛ける。精度 ~0.07′。

### 既定パラメータと出力

- `Standard` 既定: P = 1010 hPa, T = 10 ℃（呼出側で上書き可）。
- 通知用途では**幾何学的高度と見かけ高度の両方**を返す（conventions §7）。`LocalContact.sun_altitude` は既定で幾何学的、見かけは別フィールド/オプション（api-draft 反映は要追記）。
- 観測者標高による地平線伏角（必要時）: `dip(arcmin) ≈ 1.76·√(h_eye_m)`（屈折込みの近似, 要確認の出典）。既定はジオメトリのみ、伏角は明示オプション。

## C2. 日の出・日没と可視性（`Visibility`）

接触時刻が存在しても、太陽が地平線下なら観測不能。各接触点の太陽高度から可視性を決める。

### 地平しきい値

- **太陽中心の見かけ高度** を可視判定の基準量とする（`RefractionModel` に従う）。
- 標準的な「日の出/日没」（上縁が地平）: 太陽中心の真高度
  ```
  h_set = −( SD_sun + R_horizon + dip )                … (C2.1)
  ```
  - SD_sun ≈ 16′（視半径, 距離依存。conventions §9 から各時刻算出）
  - R_horizon ≈ 34′（標準地平大気差; Standard モデルの地平値）
  - dip = 標高伏角（海面 0 のとき 0）。海面・標準条件で h_set ≈ −50′ ≈ −0.8333°。
- 接触の可視フラグ `visible_i`: 太陽中心**見かけ高度 ≥ 0**（中心が見かけ地平の上）。判定基準は明示し、幾何・見かけ両高度を併記する。

### 6 値の定義

各接触（C1, [C2], 最大, [C3], C4）の見かけ太陽高度 `h_i` と、地平通過時の昇降（`dh/dt` の符号）から決める。観測窓の端は部分食 P 接触（C1/C4）を基準とする。

| 値 | 定義 |
|---|---|
| `NotVisible` | 当該地点で日食が起きない（接触が存在しない＝部分食域外） |
| `BelowHorizon` | 接触は存在するが、最大食を含め**全接触で太陽が地平下**（観測不能） |
| `FullyVisible` | **全接触で太陽が地平上** |
| `SunriseEclipse` | C1 が地平下・太陽**上昇中**に地平を越え、食の途中から可視（日の出帯食） |
| `SunsetEclipse` | C4 までに太陽が**下降して地平下**へ、食の途中で日没（日没帯食） |
| `PartialVisible` | 上記以外で一部の接触のみ地平上（昇降が端で綺麗に切れない混在ケース） |

判定手順:
1. 各接触の見かけ高度 `h_i` と昇降符号を計算。
2. 全 `h_i > 0` → `FullyVisible`。最大食含め全 `h_i ≤ 0` → `BelowHorizon`。接触なし → `NotVisible`。
3. C1 が地平下 ∧ C4 地平上 ∧ 太陽上昇（dh/dt>0）→ `SunriseEclipse`。
4. C1 地平上 ∧ C4 地平下 ∧ 太陽下降 → `SunsetEclipse`。
5. それ以外の地平上下混在 → `PartialVisible`。
- 境界・特異（極域の白夜/極夜、地平すれすれの接線、複数回地平横断）は §9 検証の地点分類でテスト化（要確認: 多重横断の扱い）。

## C3. 種別判定の閾値（全球, algorithms §8 / ISSUE-023）

`gamma`（影軸の地心最小距離, Re 単位）と本影錐半径 `u`（= ベッセル l2 相当, 符号付き）で分類する。Meeus Ch.54 の判定基準（地球扁平を織り込んだ定数）:

```
|gamma| < 0.9972                          → 中心食（total/annular/hybrid）        … (C3.1)
0.9972 ≤ |gamma| < 0.9972 + |u|           → 非中心 皆既/金環（本影/反本影が縁に接触）
0.9972 + |u| ≤ |gamma| < 1.5433 + u       → 部分食のみ
|gamma| ≥ 1.5433 + u                      → 日食なし（影錐が地球を外す）
```

- 定数 **0.9972**（≈ 1 − 扁平による縮約）と **1.5433**（半影限界）は Meeus の採用値。**要確認**: Meeus Ch.54 原文の式番号・有効桁、および本実装の Re 定義（conventions §4.1, WGS84 a）との整合（NASA 慣習との差は accuracy §4 で記録）。
- 参考の数値域: 非中心境界は概ね |gamma|≈0.9972〜1.0266、部分/食なし境界は概ね 1.53（皆既側）〜1.57（金環側）。

### 中心食の細分（l2 の符号, conventions §9 / algorithms §5）

中心食区間にわたる本影錐半径 `l2`（= u）の符号で決める:

```
l2 < 0（観測者が本影頂点より地球側）  → 皆既 (Total)            … (C3.2)
l2 > 0（反本影＝頂点より遠い）          → 金環 (Annular)
中心食の経路上で l2 が符号反転          → ハイブリッド (Hybrid)
```

- 月半径係数 k の選択（conventions §9: IauMean 既定 / Espenak 照合切替）で皆既/金環の境界が系統的に動く。照合時は Espenak 2 値 k へ切替（accuracy §2.2 / §4）。
- 非中心の `NonCentralTotal`/`NonCentralAnnular` は (C3.1) の第2域。`SolarEclipseKind` enum は公開維持（api-draft A5）。
